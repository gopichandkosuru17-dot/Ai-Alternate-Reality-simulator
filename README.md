## 🌌 Overview

AI Alternate Reality Simulator is a cinematic web experience that generates alternate future realities from user identity, ambitions, decisions, passions, and life direction.

This is **not a prediction engine**.

Instead, it creates immersive alternate future scenarios and allows users to explore different possible versions of themselves.

The experience is designed to feel like entering a futuristic reality machine.

# ⚙ Installation

Clone repository:

```bash
git clone YOUR_REPO_URL
```

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```

Open:

```text
http://localhost:5173
```
